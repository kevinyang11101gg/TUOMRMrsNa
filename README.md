# Python电影票房数据分析系统简介

## 核心功能
- 登录注册
- 电影票房数据可视化大屏
- 电影票房数据查询
- 电影论坛
- 电影资讯浏览
- 留言消息
- 用户管理
- 电影票房管理
- 留言消息管理
- 电影论坛管理

## 适用场景
- 电影行业数据分析
- 影院运营管理
- 电影市场研究
- 消费者行为分析

## 技术特点
- 采用 Django 框架进行后端开发
- 使用 MySQL 数据库存储数据
- 前端界面基于 Vue.js 实现
- 支持数据可视化展示
- 提供用户管理和电影管理功能
